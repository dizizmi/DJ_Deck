# DJ Deck
Controls and mixes music using C++ and JUCE framework.

Functionality:

-speed of music

-volume of music

-position of track

-Cues

-Loop

-upload/drag music file

-search music library

-remove track from library

-2 songs at a time

-sound reverbs; size, damp, width, wet/dry
